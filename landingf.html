    Areas Of Speciality

    HTML-"Hypertext Markup Language"

    A standardized system for tagging text files to achieve font, color, graphic, and hyperlink effects on World Wide Web pages. A primary language necessary for a website to function proplerly, acting as the main structure for all websites.

    CSS-"Cascading Style Sheet"

    CSS is often used to format the layout of Web pages. The coding language's main purpose is to define text styles, table sizes, and other aspects of Web pages, if defined by HTML. Allowing many websites since the start of the world wide web to be personally styled.
